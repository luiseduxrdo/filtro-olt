Filtro OLT

Siga os passos abaixo para configurar o servidor local

1

Copiar o comando

Clique em "Copiar" para copiar o comando que inicia o servidor local.

powershell -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/luiseduxrdo/filtro-olt/master/servidor.ps1 | iex"
2

Abrir o PowerShell e colar

Pressione Win+R, digite powershell e pressione Enter.

Na janela que abrir, cole o comando com Ctrl+V e pressione Enter.

Uma janela do servidor vai aparecer — deixe-a aberta enquanto usar o Filtro OLT.

Requisito: estar conectado a VPN da empresa.
3

Comecar a usar

Volte aqui e clique em "Testar conexao". Se o indicador ficar verde, esta tudo pronto!

sem teste

ONU DESCRIPTION onu description show gpon X
